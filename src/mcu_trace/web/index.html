ℹ️ 关于

MCU TRACE 工具 v0.2.0

功能：自动解析 MCU trace 日志，可视化电源管理生命周期曲线

核心能力：

  • 📥 一键导入加密 .enc 日志（自动调 hsaedecrypt 解密）
  • 🔍 关键字 / 错误码检索 + 分类高亮
  • 📊 PSM 电源状态机时序识别（10 状态 + 异常跳转检测）
  • 📊 SoC 系统模式（OFF/STANDBY/STR/NORMAL）时间线
  • ⚡ 电压时间曲线（支持自定义电压关键字正则）
  • 🔄 MCU 复位事件提取（按硬件枚举映射）
  • 📄 一键导出含图表 + 原始节选的 HTML 报告

作者：mafuxuan <mafuxuan@example.com>

技术栈：Python 3.12 + pywebview 4 + matplotlib + ECharts 5
